# Form 3 English Content Schema

## Purpose

This document defines a platform-ready `Form 3 English` curriculum structure for an AI-first, tutor-supported tuition system aligned to Malaysia lower secondary learning.

It extends the Form 1 and Form 2 English pathway into stronger:

- evidence-based comprehension
- purpose and attitude analysis
- summary and text structure control
- editing and expression accuracy
- register-sensitive vocabulary
- multi-paragraph and guided writing

This is a product blueprint, not a textbook chapter list.

## Scope

Subject:

- English

Level:

- Form 3

Primary instructional model:

- AI handles daily reading analysis, language practice, vocabulary precision, and guided writing support
- Human tutors step in for explanation, intervention, writing coaching, and progress review

---

## 1. Domain Map

| Domain Code | Domain Name | Why It Exists |
|---|---|---|
| `A` | Reading and Analytical Response | Move students toward evidence-based interpretation, summary, and text-structure awareness |
| `B` | Grammar, Editing and Expression Control | Strengthen sentence accuracy, reformulation, and editing for clearer written expression |
| `C` | Vocabulary, Register and Meaning | Improve precision through figurative meaning, register choice, and stronger lexical control |
| `D` | Extended and Functional Writing | Build structured multi-paragraph responses and better fit-for-purpose guided writing |

---

## 2. Topic Register

| Topic Code | Topic Name | Summary |
|---|---|---|
| `A1` | Evidence-Based Comprehension | Use direct textual evidence more carefully when answering reading questions |
| `A2` | Attitude, Purpose and Message | Infer writer attitude, intention, and key message with stronger justification |
| `A3` | Summary, Sequence and Text Structure | Track sequence and structure to identify and summarise key information accurately |
| `B1` | Sentence Accuracy and Error Correction | Strengthen editing accuracy through sentence-level correction and rule awareness |
| `B2` | Reported Meaning and Question Forms | Handle reporting, questioning, and practical meaning shifts more accurately |
| `B3` | Clause Control and Editing for Flow | Improve clarity and writing flow through stronger clause use and editing choices |
| `C1` | Figurative and Implied Meaning | Interpret non-literal and implied meaning more accurately in reading |
| `C2` | Register, Word Form and Precision | Choose words that fit audience, register, and intended meaning more precisely |
| `D1` | Multi-Paragraph Response Writing | Organise and develop ideas across multiple connected paragraphs |
| `D2` | Guided Extended and Functional Writing | Produce stronger guided and practical writing with clearer content fit and structure |

## 3. Mastery Design

Each topic is broken into `4` mastery nodes:

1. `Recognition`
2. `Procedure`
3. `Procedure / transfer`
4. `Application`

This keeps Form 3 English aligned with the platform-wide mastery model already used in mathematics and earlier English levels.

### Example: `A2 Attitude, Purpose and Message`

- `E3R2-01` Recognise signals of attitude and purpose
- `E3R2-02` Infer purpose and message from text clues
- `E3R2-03` Support attitude or purpose judgments
- `E3R2-04` Explain attitude, purpose, and message in a short response

### Example: `B3 Clause Control and Editing for Flow`

- `E3G3-01` Recognise clause-level flow and control issues
- `E3G3-02` Improve clause control in connected writing
- `E3G3-03` Edit sentence flow with better clause choices
- `E3G3-04` Edit a short text for clause flow and expression

### Example: `D1 Multi-Paragraph Response Writing`

- `E3W1-01` Recognise multi-paragraph response structure
- `E3W1-02` Plan paragraph progression across a response
- `E3W1-03` Develop support and transitions across paragraphs
- `E3W1-04` Write a structured multi-paragraph response

---

## 4. Recommended Learning Flow

For daily study, the AI tutor should sequence Form 3 English like this:

1. Reading or language warm-up
2. Micro lesson on the current mastery node
3. Guided practice
4. Short independent check
5. Revision recommendation
6. Tutor intervention if stuck

### AI tutor should prioritise:

- evidence before opinion in reading responses
- editing accuracy before stylistic variety
- purpose and audience before longer writing output
- precision and register, not only “close enough” vocabulary

---

## 5. Common Misconception Families

The current Form 3 blueprint seeds these high-value misconception groups:

- `answer-without-text-proof`
- `purpose-as-topic-confusion`
- `summary-copies-too-much`
- `editing-without-rule-awareness`
- `meaning-shift-in-reformulation`
- `flow-improves-but-clarity-drops`
- `literal-reading-of-figurative-language`
- `wrong-register-right-meaning`
- `paragraphs-without-progression`
- `task-fit-without-depth`

These are designed to drive:

- AI feedback wording
- tutor intervention rules
- parent-facing progress summaries

---

## 6. Tutor Intervention Points

Tutors should be alerted when students show patterns like:

- reading answers without textual support
- confusion between topic, purpose, and attitude
- weak summarising because too much text is copied
- sentence corrections done by guess rather than rule awareness
- reformulations that change meaning accidentally
- longer writing with weak progression across paragraphs
- vocabulary choices that are accurate in meaning but wrong in register

Tutor actions inside the platform should include:

- assign mini revision path
- unlock reading-evidence support
- model one improved paragraph or edited response
- leave a parent-visible support note
- narrow the next writing task before widening again

---

## 7. Parent Visibility

For Form 3 English, parent-facing progress should highlight:

- evidence-based reading skill
- editing and sentence accuracy
- writing organisation
- weak communication area
- next focus

### Example parent summary

- Current focus: `Attitude, Purpose and Message`
- Improved this week: `Evidence-Based Comprehension`
- Needs support: `Multi-paragraph response organisation`
- Next step: `Support every reading answer with a specific text clue`

---

## 8. MVP Build Order

Recommended build order for authored Form 3 English content:

1. `A1 Evidence-Based Comprehension`
2. `A2 Attitude, Purpose and Message`
3. `B1 Sentence Accuracy and Error Correction`
4. `B3 Clause Control and Editing for Flow`
5. `D1 Multi-Paragraph Response Writing`
6. `D2 Guided Extended and Functional Writing`
7. `A3 Summary, Sequence and Text Structure`
8. `C2 Register, Word Form and Precision`
9. `B2 Reported Meaning and Question Forms`
10. `C1 Figurative and Implied Meaning`

This order gets the platform to a useful Form 3 core first:

- stronger reading justification
- cleaner editing
- clearer multi-paragraph writing

before widening the full vocabulary and advanced interpretation layer.

---

## 9. Summary

`Form 3 English` in this platform should not be a loose collection of reading questions and grammar drills.

It should behave like a structured mastery system:

- 4 domains
- 10 topics
- 40 mastery nodes
- linked misconception rules
- tutor intervention hooks
- parent-visible progress language

This creates a clean lower secondary progression from:

- Form 1 foundations
- Form 2 control and coherence
- into Form 3 analytical reading, editing, and extended response writing
