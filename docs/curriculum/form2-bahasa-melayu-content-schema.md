# Bahasa Melayu Form 2 Content Schema

## Scope

- Subject: `BM-KSSM`
- Level: `FORM-2`
- Goal: move students from basic comprehension and sentence control into clearer interpretation, more controlled language use, and more developed writing.

## Domain Architecture

### A. Interaksi Lisan dan Penyampaian Maklumat
- `A1` Mendengar dan Menilai Maklumat
- `A2` Bertutur, Memberi Pendapat dan Hujah Ringkas

### B. Membaca, Menilai dan Mentafsir
- `B1` Mengenal Pasti Idea Utama, Butiran dan Bukti Teks
- `B2` Inferens, Sudut Pandang dan Maksud Tersirat
- `B3` Ringkasan dan Perbandingan Maklumat

### C. Tatabahasa, Kosa Kata dan Wacana
- `C1` Penyuntingan Ayat dan Ketepatan Tatabahasa
- `C2` Imbuhan, Apitan dan Penanda Wacana
- `C3` Kosa Kata Tepat, Peribahasa dan Gaya Bahasa

### D. Penulisan Berstruktur dan Fungsional
- `D1` Membina Perenggan Huraian dan Kohesi
- `D2` Penulisan Berpandu, Format dan Kejelasan Isi

## Mastery Model

Each topic is split into four nodes:
- `Recognition`
- `Procedure`
- `Procedure`
- `Application`

This keeps the BM learning flow parallel with Mathematics and English so dashboards, AI tutoring, and tutor interventions can reuse the same platform logic.

## Example Node Flow

### `B2` Inferens, Sudut Pandang dan Maksud Tersirat
- `BM2R2-01` Mengesan petunjuk sudut pandang dan maksud tersirat
- `BM2R2-02` Membina inferens yang disokong teks
- `BM2R2-03` Menjelaskan sudut pandang atau nada mudah
- `BM2R2-04` Menulis respons inferens yang beralasan

### `D1` Membina Perenggan Huraian dan Kohesi
- `BM2W1-01` Mengenal pasti struktur perenggan huraian
- `BM2W1-02` Mengembangkan isi dalam perenggan
- `BM2W1-03` Menggunakan kohesi untuk melancarkan perenggan
- `BM2W1-04` Menulis perenggan huraian yang tersusun

## Misconception Families

The blueprint currently includes ten misconception families, including:
- fakta didengar tetapi tidak ditapis
- pendapat tanpa alasan jelas
- bukti teks tidak tepat
- sudut pandang disalah tafsir
- perbandingan jadi senarai
- edit pada tempat salah
- penanda wacana hanya tempelan
- ungkapan sesuai makna tetapi tidak sesuai konteks
- huraian ada tetapi tidak berkembang
- format tepat tetapi tugasan tidak benar-benar dijawab

## Tutor Intervention Points

Tutor intervention is recommended when:
- inferens is repeatedly made without text support
- paragraph writing has topic sentences but weak development
- grammar editing fixes surface words but misses the real error source
- guided writing follows format but still fails task intent

## Parent Visibility

Parent-facing reports for Form 2 BM should highlight:
- current support focus
- reading interpretation progress
- writing coherence progress
- language control issues that need tutor follow-up
- next revision topic

## MVP Build Order

Recommended rollout order:
1. `B1` Mengenal Pasti Idea Utama, Butiran dan Bukti Teks
2. `C1` Penyuntingan Ayat dan Ketepatan Tatabahasa
3. `D1` Membina Perenggan Huraian dan Kohesi
4. `B2` Inferens, Sudut Pandang dan Maksud Tersirat
5. `D2` Penulisan Berpandu, Format dan Kejelasan Isi

`Bahasa Melayu Form 2` is now structured as a full curriculum blueprint with:
