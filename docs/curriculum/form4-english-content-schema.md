# Form 4 English Content Schema

## Purpose

This document defines a platform-ready `Form 4 English` curriculum structure for an AI-first, tutor-supported tuition system aligned to Malaysia upper secondary learning.

It extends the lower secondary English pathway into stronger:

- evidence synthesis across texts
- viewpoint and writer-craft analysis
- editing for accuracy and style
- reformulation and argument control
- register-sensitive vocabulary precision
- longer essay and exam-oriented writing

This is a product blueprint, not a textbook chapter list.

## Scope

Subject:

- English

Level:

- Form 4

Primary instructional model:

- AI handles daily reading analysis, editing practice, vocabulary refinement, and writing rehearsal
- Human tutors step in for interpretation coaching, essay feedback, intervention planning, and exam preparation

---

## 1. Domain Map

| Domain Code | Domain Name | Why It Exists |
|---|---|---|
| `A` | Reading and Critical Response | Develop stronger synthesis, interpretation, and writer-craft analysis in upper secondary reading |
| `B` | Grammar and Expression for Upper Secondary | Improve editing, sentence transformation, and controlled language use for more demanding tasks |
| `C` | Vocabulary, Style and Interpretation | Build connotation awareness, register control, and more precise vocabulary choices |
| `D` | Extended and Exam-Oriented Writing | Prepare students for longer essays and practical writing tasks with clearer structure and task control |

---

## 2. Topic Register

| Topic Code | Topic Name | Summary |
|---|---|---|
| `A1` | Evidence Synthesis and Comparison | Compare and combine evidence from related texts to build stronger reading responses |
| `A2` | Viewpoint, Tone and Writer Craft | Analyse tone, viewpoint, and how language choices shape reader response |
| `A3` | Summary, Evaluation and Text Links | Summarise accurately, judge key ideas, and explain links within or across texts |
| `B1` | Editing for Accuracy and Style | Improve both correctness and style when revising sentences and short passages |
| `B2` | Sentence Transformation and Reformulation | Transform and reformulate meaning without losing accuracy or task fit |
| `B3` | Argument Language and Cohesive Control | Use cohesive devices and argument language to build clearer logical flow |
| `C1` | Connotation and Figurative Effect | Interpret connotation and figurative language more precisely in analysis |
| `C2` | Register, Audience and Precision | Choose language that fits tone, audience, formality, and intended effect |
| `D1` | Extended Response and Essay Structure | Plan and write longer responses with clear structure, support, and progression |
| `D2` | Exam Task and Functional Writing | Produce stronger exam-style guided and functional writing with better task control |

## 3. Mastery Design

Each topic is broken into `4` mastery nodes:

1. `Recognition`
2. `Procedure`
3. `Procedure / transfer`
4. `Application`

This keeps Form 4 English aligned with the platform-wide mastery model already used in mathematics and earlier English levels.

### Example: `A2 Viewpoint, Tone and Writer Craft`

- `E4R2-01` Recognise writer-craft signals
- `E4R2-02` Infer tone and viewpoint through writer choices
- `E4R2-03` Explain how writer craft shapes meaning
- `E4R2-04` Respond critically to viewpoint and writer craft

### Example: `B3 Argument Language and Cohesive Control`

- `E4G3-01` Recognise argument language and cohesion signals
- `E4G3-02` Use cohesive devices to organise argument
- `E4G3-03` Strengthen flow in an argument paragraph
- `E4G3-04` Apply cohesive argument language in writing

### Example: `D1 Extended Response and Essay Structure`

- `E4W1-01` Recognise essay-level structure and progression
- `E4W1-02` Plan an extended response with clear support
- `E4W1-03` Develop paragraph progression across an essay
- `E4W1-04` Write a structured extended response

---

## 4. Recommended Learning Flow

For daily study, the AI tutor should sequence Form 4 English like this:

1. Reading or editing warm-up
2. Micro lesson on the current mastery node
3. Guided practice
4. Short independent check
5. Revision recommendation
6. Tutor intervention if needed

### AI tutor should prioritise:

- synthesis before superficial comparison
- effect and reader response, not feature-spotting alone
- meaning-preserving reformulation
- structured essay planning before longer writing output
- register and audience fit, not just “correct” vocabulary

---

## 5. Common Misconception Families

The current Form 4 blueprint seeds these high-value misconception groups:

- `two-text-answer-one-source-only`
- `technique-name-without-effect`
- `summary-with-commentary-mix`
- `correct-but-still-awkward`
- `transformation-breaks-meaning`
- `connector-rich-logic-weak`
- `literal-meaning-overrides-effect`
- `tone-does-not-match-reader`
- `essay-has-paragraphs-no-thesis-thread`
- `task-format-correct-content-underpowered`

These are designed to drive:

- AI feedback wording
- tutor intervention rules
- parent-facing progress summaries

---

## 6. Tutor Intervention Points

Tutors should be alerted when students show patterns like:

- comparison answers using only one source
- identifying techniques without discussing effect
- summarising and commenting in the same sentence without control
- editing that fixes grammar but not clarity or style
- sentence transformations that change meaning
- cohesive markers used without a real logical structure
- vocabulary that is understandable but wrong for audience or formality
- essays with paragraphs but weak thesis progression

Tutor actions inside the platform should include:

- assign a narrow revision path
- model one strong analytical response
- demonstrate one better paragraph plan
- give essay-structure feedback
- leave a parent-visible support note

---

## 7. Parent Visibility

For Form 4 English, parent-facing progress should highlight:

- reading analysis strength
- editing accuracy
- essay organisation
- weak communication area
- next focus

### Example parent summary

- Current focus: `Viewpoint, Tone and Writer Craft`
- Improved this week: `Evidence Synthesis and Comparison`
- Needs support: `Essay progression and support`
- Next step: `Explain how language choice affects reader response`

## 8. MVP Build Order

Recommended build order for authored Form 4 English content:

1. `A1 Evidence Synthesis and Comparison`
2. `A2 Viewpoint, Tone and Writer Craft`
3. `B1 Editing for Accuracy and Style`
4. `B3 Argument Language and Cohesive Control`
5. `D1 Extended Response and Essay Structure`
6. `D2 Exam Task and Functional Writing`
7. `B2 Sentence Transformation and Reformulation`
8. `C2 Register, Audience and Precision`
9. `A3 Summary, Evaluation and Text Links`
10. `C1 Connotation and Figurative Effect`

This order gets the platform to a useful Form 4 core first:

- stronger evidence-based reading
- cleaner editing and argument control
- clearer essay structure

before widening the full stylistic and interpretive layer.

---

## 9. Summary

`Form 4 English` in this platform should not be a loose mix of essays, grammar drills, and comprehension questions.

It should behave like a structured mastery system:

- 4 domains
- 10 topics
- 40 mastery nodes
- linked misconception rules
- tutor intervention hooks
- parent-visible progress language

This creates a clean upper secondary progression from:

- Form 1 foundations
- Form 2 control and coherence
- Form 3 analytical reading and extended response
- into Form 4 synthesis, style, and essay-ready writing
