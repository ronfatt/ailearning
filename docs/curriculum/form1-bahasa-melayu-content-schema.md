# Form 1 Bahasa Melayu Content Schema

## Purpose

This document defines a platform-ready `Form 1 Bahasa Melayu` curriculum structure for an AI-first, tutor-supported tuition system aligned to Malaysia lower secondary KSSM learning goals.

It is designed to support:

- listening and oral response
- literal and inferential reading
- sentence and grammar control
- vocabulary, peribahasa, and language appreciation
- guided writing and functional response

This is a product blueprint, not a textbook chapter list.

## Scope

Subject:

- Bahasa Melayu

Level:

- Form 1

Primary instructional model:

- AI handles daily reading, language drills, structured response, and writing rehearsal
- Human tutors step in for explanation, speaking support, intervention planning, and writing feedback

---

## 1. Domain Map

| Domain Code | Domain Name | Why It Exists |
|---|---|---|
| `A` | Mendengar, Bertutur dan Respons Lisan | Build listening accuracy and basic spoken response for school and daily contexts |
| `B` | Membaca dan Memahami | Strengthen literal understanding, inferens awal, and text response |
| `C` | Tatabahasa, Kosa Kata dan Keindahan Bahasa | Build grammatical control, vocabulary growth, and early appreciation of language use |
| `D` | Penulisan dan Respons Komunikatif | Develop sentence writing, paragraphing, and guided / functional writing |

## 3. Mastery Design

Each topic is broken into `4` mastery nodes:

1. `Recognition`
2. `Procedure`
3. `Procedure / transfer`
4. `Application`

This keeps Form 1 Bahasa Melayu aligned with the platform-wide mastery model already used in mathematics and English.

### Example: `B2 Inferens, Makna Konteks dan Urutan`

- `BM1R2-01` Mengenal pasti petunjuk inferens
- `BM1R2-02` Menentukan makna perkataan melalui konteks
- `BM1R2-03` Mengesan urutan idea atau peristiwa
- `BM1R2-04` Membuat inferens mudah daripada teks

### Example: `C2 Imbuhan, Golongan Kata dan Penggunaan`

- `BM1G2-01` Mengenal pasti golongan kata dan imbuhan
- `BM1G2-02` Menggunakan imbuhan dengan betul
- `BM1G2-03` Memilih golongan kata yang sesuai
- `BM1G2-04` Mengaplikasi imbuhan dan golongan kata dalam penulisan

### Example: `D2 Penulisan Berpandu dan Respons Fungsional`

- `BM1W2-01` Mengenal pasti kehendak tugasan penulisan
- `BM1W2-02` Merancang isi dan susunan penulisan
- `BM1W2-03` Menulis respons mengikut format dan tujuan
- `BM1W2-04` Menghasilkan penulisan berpandu yang lengkap

## 4. Recommended Learning Flow

For daily study, the AI tutor should sequence Form 1 Bahasa Melayu like this:

1. Short comprehension or language warm-up
2. Micro lesson on the current mastery node
3. Guided practice
4. Short independent response
5. Revision recommendation
6. Tutor intervention if needed

### AI tutor should prioritise:

- understanding before memorising
- evidence from text before unsupported inferens
- grammar in real sentence use, not isolated rule recall only
- guided paragraph structure before longer writing
- task fit and format in functional responses

## 5. Common Misconception Families

The current Form 1 blueprint seeds these high-value misconception groups:

- `arahan-dan-butiran-bercampur`
- `respons-lisan-tidak-teratur`
- `isi-utama-disalah-anggar`
- `inferens-terlalu-luar-teks`
- `ringkasan-menyalin-terus`
- `subjek-predikat-tidak-seimbang`
- `imbuhan-salah-konteks`
- `peribahasa-hafal-tidak-tepat-guna`
- `perenggan-himpunan-ayat-sahaja`
- `format-betul-isi-lemah`

These are designed to drive:

- AI feedback wording
- tutor intervention rules
- parent-facing progress summaries

---

## 6. Tutor Intervention Points

Tutors should be alerted when students show patterns like:

- hearing only part of an instruction
- giving oral responses without clear structure
- confusing main idea with supporting detail
- making inferens not supported by text evidence
- copying directly into ringkasan
- writing grammatically incomplete sentences
- choosing the wrong imbuhan for context
- using peribahasa in the wrong situation
- producing paragraphs without one clear idea thread
- following format but not answering task content strongly enough

Tutor actions inside the platform should include:

- assign a narrow revision path
- model one stronger oral or written response
- rebuild sentence structure with live coaching
- give paragraph-planning support
- leave a parent-visible support note

---

## 7. Parent Visibility

For Form 1 Bahasa Melayu, parent-facing progress should highlight:

- reading comprehension
- grammar and sentence control
- vocabulary / language use
- writing clarity
- next focus

### Example parent summary

- Current focus: `Inferens, Makna Konteks dan Urutan`
- Improved this week: `Pembinaan Ayat dan Tatabahasa Asas`
- Needs support: `Penulisan berpandu`
- Next step: `Gunakan bukti teks sebelum menulis inferens`

## 8. MVP Build Order

Recommended build order for authored Form 1 Bahasa Melayu content:

1. `B1 Memahami Idea Utama dan Maklumat Literal`
2. `C1 Pembinaan Ayat dan Tatabahasa Asas`
3. `D1 Membina Ayat dan Perenggan`
4. `B2 Inferens, Makna Konteks dan Urutan`
5. `C2 Imbuhan, Golongan Kata dan Penggunaan`
6. `D2 Penulisan Berpandu dan Respons Fungsional`
7. `A1 Mendengar Maklumat dan Arahan`
8. `A2 Bertutur untuk Maklumat dan Respons`
9. `B3 Ringkasan Ringkas dan Respons Teks`
10. `C3 Peribahasa, Kosa Kata dan Keindahan Bahasa`

This order gets the platform to a useful Form 1 BM core first:

- reading comprehension
- sentence and grammar control
- paragraph writing

before widening speaking, ringkasan, and language-appreciation layers.

---

## 9. Summary

`Form 1 Bahasa Melayu` in this platform should not become a loose mix of grammar notes, karangan prompts, and comprehension questions.

It should behave like a structured mastery system:

- 4 domains
- 10 topics
- 40 mastery nodes
- linked misconception rules
- tutor intervention hooks
- parent-visible progress language

This gives the platform a clear Bahasa Melayu foundation that can later scale into:

- Form 2 language control and writing growth
- Form 3 analytical reading and stronger response tasks
- upper secondary BM pathways
